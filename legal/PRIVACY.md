# プライバシーポリシー (Privacy Policy)

「やさしい読み上げ (read-aloud-easy)」(以下「本拡張機能」) は、ユーザーのプライバシーを最優先に考え、設計されています。

## 個人情報の収集について
本拡張機能は、氏名、メールアドレス、電話番号、ブラウザの閲覧履歴などの個人情報を一切収集しません。

## データの外部送信について
本拡張機能は、収集した設定データや利用状況を外部のサーバーに送信することはありません。すべての拡張機能データはユーザーのローカルデバイス (chrome.storage.local) 内でのみ保存・管理されます。ユーザーが Premium へのアップグレードを選択した場合のみ、Stripe の支払いページを新しいタブで開きます。支払い情報は Stripe 側で処理され、本拡張機能は保存済みの設定、ブックマーク、選択テキスト、閲覧履歴を Stripe に送信しません。

## 解析ツール等の使用について
本拡張機能では、Google Analytics などのアクセス解析ツールや広告配信ツールを使用していません。

## お問い合わせ
本プライバシーポリシーに関するお問い合わせは、GitHub リポジトリの Issue を通じてご連絡ください。

# Privacy Policy

"read-aloud-easy" (hereinafter referred to as "this Extension") is designed with the user's privacy as the top priority.

## Collection of Personal Information
This Extension does not collect any personal information such as name, email address, phone number, or browsing history.

## External Transmission of Data
This Extension does not transmit any collected setting data or usage status to external servers. All extension data is saved and managed only within the user's local device (chrome.storage.local). Only when the user chooses to upgrade to Premium, the Extension opens a Stripe payment page in a new tab. Payment information is handled by Stripe, and the Extension does not send saved settings, bookmarks, selected text, or browsing history to Stripe.

## Use of Analysis Tools, etc.
This Extension does not use access analysis tools such as Google Analytics or advertising delivery tools.

## Contact Us
For inquiries regarding this Privacy Policy, please contact us through the Issues section of our GitHub repository.
